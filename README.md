# predictive-maintenance-classification
Machine Learning project for predicting machine failures using feature engineering, SMOTE, PCA, feature selection, and ensemble models including Random Forest, XGBoost, and CatBoost.
## 📌 Overview

Predictive Maintenance is one of the most impactful applications of Machine Learning in Industry 4.0.

This project predicts machine failures using sensor and operational data. Multiple machine learning models are trained and evaluated to identify potential equipment failures before they occur, helping reduce downtime and maintenance costs.

---

## 🎯 Objective

Build a classification model capable of predicting whether a machine is likely to fail based on its operating conditions and sensor measurements.

---

## 📊 Dataset

The dataset contains machine operating parameters and sensor readings.

Target Variable:

- Machine_failure
  - 0 → No Failure
  - 1 → Failure
